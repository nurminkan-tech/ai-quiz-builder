# Шырақ — AI Квиз Құраушы 🔥

AI-қуатталған интерактивті сабақ ойын генератору. Учитель тақырыпты жазады — AI төрт түрлі ойын құрастырады!

## Ұзындықтар

Шырақ (**Shyryq**) — қазақстандық мектептің оқушыларына арналған цифрлық құрал. Мұғалім бір сабақтың тақырыбын енгізеді, жасанды интеллект сол тәрбиелік мақсатқа сай төрт түрлі интерактивті ойын құрастырады:

- **⚡ Kahoot-квиз** — уақыты сұрақтар мен төрт нұсқадан таңдау
- **🔗 Wordwall-сәйкестендіру** — терминдер мен анықтамаларды қосу
- **🎯 Дорба (Jeopardy)** — балл ұпайын сұрақ таңдап ойнау
- **▦ Кроссворд** — сөздерді табу және толтыру

## Ерекшеліктер

✨ **Қарапайым және құлыңды:**
- Ешқандай санды түйнеге басу, мәзір навигация жоқ
- Оқушылар түсінетін қазақ тіліндегі интерфейс
- Мобильді құрылғыларда сағына отырады

🤖 **AI-ге негізделген:**
- Claude (Anthropic) AI модель ойындар құрастырады
- Әр басқанда жаңа, өзгеше сұрақтар
- Пәнге және сыныпқа сай мазмұн

🎮 **Төрт түрлі ойын форматы:**
- Кездейсоқ режимі — әр сабақта басқа ойын
- Немесе сіздің таңдағаныңызды таңдаңыз
- Жеке, топтық немесе сыныптық режимдер

## Технология

- **Frontend:** Таза JavaScript + CSS (бөлінген компоненттер)
- **Backend:** Node.js (Vercel Serverless Functions)
- **API:** Anthropic Claude API
- **Deployment:** Vercel (өрі-ешсіз)

## Іске қосу

### 1. GitHub репозиторийны клон қылу

```bash
git clone https://github.com/nurminkan-tech/ai-quiz-builder.git
cd ai-quiz-builder
```

### 2. Vercel-де орнату

Ең қарапайым әдіс — Vercel-ге тікелей байланыстыру:

1. [vercel.com](https://vercel.com) қараңыз
2. **"New Project"** басыңыз
3. GitHub репозиториясын таңдаңыз
4. **Environment Variables** бөлімінде қосыңыз:
   ```
   ANTHROPIC_API_KEY = sk-ant-...
   ```
5. **Deploy** басыңыз ✅

### 3. Орын сыналу (Local)

```bash
# Node.js 18+ орнатсаңыз бола
node -v  # v18+ ба тексеріңіз

# Бәлкі localhost қосу керек болса:
# index.html-ді браузерде ашыңыз (file:// схемасында)
# немесе Python сервері іске қосыңыз:
python -m http.server 3000
```

Сонда `http://localhost:3000` қараңыз.

## API Құлақ

### POST `/api/generate`

**Сұраныс:**
```json
{
  "prompt": "Сен қазақстандық мектеп мұғаліміне арналған сабақ ойынын құрастырушысың..."
}
```

**Жауап:**
```json
{
  "text": "{\"qs\": [{\"q\": \"сұрақ\", \"o\": [...], \"a\": 0}]}"
}
```

Қателік болса:
```json
{
  "error": "Сервер қатесі: ..."
}
```

## Барлық Форматтар туралы

### 1. Kahoot-квиз ⚡
- 6 сұрақ, әр сұрақта 4 нұсқа
- 20 секунд таймер
- Балл есептеу

### 2. Wordwall-сәйкестендіру 🔗
- 6 термин-анықтама жұбы
- Тармақтарын қосу арқылы ойну
- Визуалды кері сигналдар

### 3. Дорба (Jeopardy) 🎯
- 3 санат × 3 сұрақ
- Балл мәндері: 100, 200, 300
- 2 топтың ұпайы тұрғысында

### 4. Кроссворд ▦
- 6 сөз (минимум) көлденең + тік
- Анықтамалар әр сөзге
- Тексеру функциясы

## Қателер + Шешімі

| Қателік | Себебі | Шешімі |
|---------|--------|--------|
| "ANTHROPIC_API_KEY орнатылмаған" | Vercel-де env құлпы жоқ | Vercel Settings → Environment Variables қосыңыз |
| "JSON парсе ошибка" | AI таза JSON қайтарған жоқ | Prompt өндіктен бақылаңыз, Claude моделін өндіктен ұсыныңыз |
| "CORS қателігі" | Frontend өзге источник қызмет ету | Vercel CORS аутоматты құрыла береді |
| "Қателік: Сервер қатесі" | API толыс қызмет түзетен жоқ | API логын Vercel платформасында қараңыз |
